## What I'm building

Right now, I’m focused on **Somi**, a GenAI platform built around memory, personalization, and more natural long-term AI interaction.

The goal is to make conversations feel more continuous and useful by allowing the system to remember context, retrieve relevant memories, and route between faster responses and deeper reasoning when needed.

Some of the main things I’m working on include:

* Dual-mode LLM routing for fast vs deeper reasoning responses
* Semantic memory retrieval using `pgvector`
* Multi-turn conversation handling
* Layered safety checks with classification and policy gating
* API regression tests to prevent behavioural drift
* Multi-tenant backend security using Postgres RLS

I’m also continuing to sharpen my skills around production AI systems, especially retrieval quality, agent workflows, evaluation, backend reliability, and building AI products that feel intuitive to use.
